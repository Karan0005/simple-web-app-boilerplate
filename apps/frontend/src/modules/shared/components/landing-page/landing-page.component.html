Welcome to the Ultimate Angular & Nest.js Boilerplate

A powerful, modern boilerplate based on Angular and Nest.js to streamline your development experience and scale with ease.

Why Choose This Boilerplate?

Environment Parametrization

Supports multiple environments like Local, Dev, UAT, and Prod, ensuring smooth transitions across environments with configurable parameters.

Comprehensive Testing Suite

Comes with complete unit tests and end-to-end test cases, making sure your project runs reliably and consistently across environments.

Dockerized Deployment

Pre-configured Docker setup allows you to easily deploy your application in any environment, from development to production.

CI/CD Integration

Seamless GitHub CI/CD pipeline integration ensures continuous integration and delivery with minimal configuration.

Strict ESLint & VS Code Integration

Follows strict ESLint rules with recommended VS Code configurations and extensions to keep your codebase clean and consistent.

Smart Package Management

Handy scripts in package.json for easy project management, including upgrade scripts and validation checks for Node.js & npm.
